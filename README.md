# Chatbot Demo

Demo chatbots for Nordic AI clients.

## Setup

```bash
npm install
```

## Run

```bash
npm start
```

Or with auto-reload:

```bash
npm run dev
```

## Usage

Visit `http://localhost:3000/:client` where `:client` is the folder name in `/demos`.

Example: `http://localhost:3000/arena-treningssenter`
